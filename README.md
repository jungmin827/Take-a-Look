# Take a Look
> **"당신은 왜 그것을 파게 되었나요?"** > 소비 기록을 넘어, 내면의 깊은 취향을 탐구하고 전파하는 취향 에세이 플랫폼입니다.

---

## 📌 Project Vision
기존의 소셜 미디어가 '무엇을 소비했는가(**Tracking**)'에 집중할 때, 비주류는 **'왜 사랑하는가(Why)'**에 집중합니다.
머니그라피 '비주류 초대석' 감성에 공감하는 이들을 위해, 단순한 기록을 넘어 취향의 동기를 발견하고 전파하는 아카이빙 공간을 지향합니다.

---

## ✨ 핵심 경험 (Phase A MVP)
현재는 독자에게 최상의 읽기 경험을 제공하고 콘텐츠의 힘을 검증하는 것에 집중합니다.

### 🖼️ 시각적 탐색
* **Letterboxd 스타일 피드:** 3열 반응형 그리드로 구성된 피드를 통해 각 취향의 분위기를 시각적으로 먼저 전달합니다.
* **태그 시스템:** 딱딱한 카테고리 대신 유연한 '태그 칩'을 사용하여 파편화된 개인의 취향을 연결합니다.

### 📖 몰입하는 읽기
* **에세이 중심 레이아웃:** 텍스트와 이미지의 조화를 통해 필자의 깊은 '덕질' 동기를 온전히 느낄 수 있도록 설계되었습니다.
* **모바일 퍼스트:** 이동 중에도 쾌적하게 읽을 수 있는 최적화된 타이포그래피와 사용자 경험을 제공합니다.

### 🔗 발견과 전파
* **공유 최적화:** 카카오톡 등 외부 공유 시 취향이 매력적으로 돋보일 수 있도록 전용 OG 이미지와 메타데이터를 관리합니다.
* **SEO 기본 세팅:** 검색을 통해 새로운 취향을 찾는 이들에게 더 잘 닿을 수 있도록 구조화된 데이터를 적용합니다.

---

## 🛠️ Service Infrastructure
서비스의 본질에 집중하기 위해 다음과 같은 기술적 기반을 선택했습니다.

| 분류 | 기술 스택 | 선택 근거 |
| :--- | :--- | :--- |
| **Framework** | **Next.js** | 정적 생성(SSG)을 통한 빠른 로딩 및 검색 엔진 최적화 |
| **Styling** | **Tailwind CSS** | 생산성 높은 반응형 디자인 구현 |
| **Content** | **MDX** | 마크다운 기반의 유연하고 풍부한 콘텐츠 표현력 |
| **Deployment** | **Vercel** | 안정적인 호스팅 및 성능 분석 환경 제공 |

---

## 🛡️ Service Philosophy
사용자의 기록이 안전하고 가치 있게 보존될 수 있도록 다음의 원칙을 준수합니다.

* **URL 안정성:** 공유 링크가 깨지는 것을 방지하기 위해 영문 로마자 슬러그 규칙을 강제합니다.
* **데이터 무결성:** 콘텐츠 발행 전 필수 정보(커버 이미지, 요약문 등)를 시스템적으로 검증하여 완성도 높은 환경을 유지합니다.
* **접근성 준수:** 모든 시각 요소에 대체 텍스트를 필수화하여 보편적인 읽기 환경을 조성합니다.
