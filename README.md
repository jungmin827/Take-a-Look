비주류 (Bijuryoo) — 취향 에세이 플랫폼
"당신은 왜 그것을 파게 되었나요?"

  


단순히 무엇을 샀는지 기록하는 것을 넘어, 내면의 깊은 취향을 탐구하고 그 이유를 전파하는 에세이 중심의 플랫폼입니다.  

📌 Project Vision
기존의 니치 소셜 미디어(Letterboxd, Goodreads 등)가 '무엇을 얼마나 소비했는가'라는 트래킹(Tracking)에 집중한다면, 비주류는 '왜 사랑하는가(Why)'라는 근본적인 질문에 집중합니다.  

머니그라피 '비주류 초대석' 콘텐츠에 열광하는 2030 세대처럼, 자신만의 독특하고 깊은 취향을 가진 사람들이 모여 서로의 '덕질' 동기를 발견하고 전파하는 아카이빙 공간을 지향합니다.  

✨ Phase A: MVP Focus
초기 모델(Phase A)의 핵심 목표는 '최상의 읽기 경험'과 '취향의 시각적 탐색'입니다.  

시각적 취향 탐색 (Letterboxd Style): 3열 반응형 그리드 피드를 통해 각 에세이가 가진 고유의 분위기를 시각적으로 먼저 전달합니다.  

에세이 본연의 가치: 복잡한 소셜 기능 대신, 텍스트와 이미지의 조화를 통해 필자가 전달하고자 하는 취향의 깊이를 온전히 느낄 수 있도록 설계되었습니다.  

유연한 분류 (Tag System): 경직된 카테고리 구조를 탈피하고, 파편화된 개인의 취향을 유연하게 연결하는 태그 시스템을 채택했습니다.  

공유 최적화: 내가 파고 있는 취향이 외부(카카오톡 등)에서도 매력적으로 보일 수 있도록 맞춤형 썸네일과 공유 환경을 제공합니다.  

🛠 Service Infrastructure
빠르고 견고한 서비스 환경을 구축하기 위해 다음의 기술적 기반을 활용합니다.  

Next.js (Pages Router): 콘텐츠 중심 서비스에 최적화된 정적 생성(SSG) 기술을 사용하여 빠른 로딩 속도를 보장합니다.  

MDX 시스템: 작성자가 텍스트 이상의 풍부한 표현력을 가질 수 있도록 마크다운 기반의 유연한 콘텐츠 관리 체계를 갖추었습니다.  

Mobile-First Design: 이동 시간 중에도 에세이에 몰입할 수 있도록 모바일 뷰포트에서의 읽기 환경을 최우선으로 고려했습니다.  

🛡 Service Philosophy (Safety & Stability)
사용자의 기록이 안전하고 가치 있게 보존될 수 있도록 내부적인 규칙을 준수합니다.  

URL 안정성: 공유 링크가 깨지는 것을 방지하기 위해 영문 로마자 기반의 고유 식별자(Slug)를 사용하여 소셜 공유의 지속성을 보장합니다.  

데이터 무결성 검증: 에세이 발행 전 필수 정보(커버 이미지, 요약문 등)를 시스템적으로 검증하여 완성도 낮은 콘텐츠 노출을 방지합니다.  

🚀 Future Roadmap
비주류는 검증된 콘텐츠 가치를 바탕으로 단계적으로 확장될 예정입니다.  

Phase A (Current)
콘텐츠 중심의 정적 에세이 플랫폼 구축  

취향 공유 환경 및 독서 경험 검증  

Phase B (Next)
커뮤니티 확장: 독자 참여를 위한 소셜 기능(댓글 및 반응) 도입  

창작 도구 고도화: 누구나 쉽게 취향을 기록할 수 있는 관리자 에디터 제공  

콘텐츠 큐레이션: 에세이가 누적됨에 따라 취향의 흐름을 보여주는 'Obsession Trails' 기능 도입 검토  

이 프로젝트는 자신의 '비주류'적인 취향을 당당하게 드러내는 모든 이들을 위해 만들어졌습니다.
